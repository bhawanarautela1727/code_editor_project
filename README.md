# code_editor_project

This project is a simple code editor that allows you to write HTML, CSS, and JavaScript code and view the live output in real-time within the same window. It provides a playground for front-end web development.

Features:
1. Edit HTML, CSS, and JavaScript in separate panels.
2. See the live output of your code immediately as you type.
3. Lightweight and easy to use.

-> Getting Started  Prerequisites
   To run the project locally, you'll need:

a. A modern web browser (e.g., Chrome, Firefox, Edge, etc.)
b. No additional software installation required.
